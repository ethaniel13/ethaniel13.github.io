1.1.1Blown to Bits Reflection

In Hal Abelson’s, Ken Ledeen’s, and Harry Lewis’ Blown to Bits the authors return
to the topic of computer changing society many times not only to emphasize their
opinion but to draw attention to the facts behind what computing does.
All information, monitoring, and data that is done through computing is never truly
private, and this is shown through an anecdote in Blown to Bits that talks of a
governor that got caught involved with prostitution simply through a monitoring
system in his bank that was built to combat organized crime and terrorism. While
this proves that computing isn’t perfect, it does show that it has potential to
create a huge impact in crime fighting and monitoring in the future. There is
already so much information being sent through various computing platforms in the
present, that there is no question that computing will continue to transform
society in the future.



1.1.2 Lightbot Reflection

Given the chance to explore the program Lightbot, I was able to see many similarities
between Lightbot and a computer; such as a computer runs off a series of inputs and
actions. While a computer has many more complex actions, it has the same general
concept as the simple program and many of the same purposes. A computer is
designed to run off commands just as lightbot does, however, lightbot is purely software
and doesn’t have the power of a computer’s hardware. Computers also have the ability to
self correct at some points where lightbot is programmed to be corrected by the user.

Duck quest is a game based around a man who lost his duck in a dungeon and now must go and
find it by looking through the rooms of the dungeon to see where his duck went. This program
was designed using scratch, and the video illustrates the use of our “doors.” Each “door”
is a separate sprite that will be hidden on any room that the door would not be on. When the
player character comes in contact with a door sprite the sprite broadcasts the command for the
room it is assigned and it loads into the next area.
